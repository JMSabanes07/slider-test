## Getting Started

1 - Clone the project

```bash
git clone https://github.com/JMSabanes07/slider-test.git
```

2 - Install the dependencies

```bash
npm i

or

yarn install
```

3 - Start the project

```bash
npm run dev

or

yarn run dev
```

4 - Go to http://localhost:3000
